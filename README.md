# worldcup2018cli
world cup cli based application.

## Installation

    $ git clone https://github.com/iqbaladinur/worldcup2018cli.git
    $ npm install
    $ npm link

## Operation
get help with running this command.

    $ livescore --help
    $ livescore --version
